module std::marker

Builtin protocols

These are built-in marker protocols that cannot be implemented by user code, even though they contain no associated functions. They are handled specially by the compiler.

protocols

Any
Matches any type
ZeroSized
Matches zero-sized types (void, named function types, empty structs, empty arrays, ...)
Primitive
Matches all primitive types
Numeric
Integer and floating point types
Integer
Integer types
FloatingPoint
Floating point types
Signed
Signed integer types (i8, i16, i32, ...)
Unsigned
Unsigned integer types (u8, u16, u32, ...)
Pointer
All pointers
Array
All (fixed-size) arrays
Tuple
All tuples
NamedFunction
All named functions (including lambdas, but excluding closures)
FunctionPointer
Function pointers
Callable
All callable objects (named functions, closures, and function pointers)
ArrayOf
Arrays with element type T
PointerOf
Pointers to T (regardless of mutability)
