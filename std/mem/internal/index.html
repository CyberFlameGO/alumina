  • fn slice_index<Ptr>(a: slice<Ptr>, idx: usize) -> Ptr
    Ptr: Pointer

    source

    Implementation for single-element indexing for slices.

    Compiler will convert slice[i] to *slice_index(slice, i). Slice indexing is bounds-checked in debug mode.

  • fn slice_range_index<Ptr, T>(a: slice<Ptr>, range: T) -> slice<Ptr>
    Ptr: Pointer

    source

    Implementation for range indexing for slices.

    Compiler will convert slice[ran..ge] to slice_index(slice, ran..ge). It is generic over all range types (.., ran.., ..ge and ran..ge)

  • fn slice_const_coerce<T>(a: slice<&mut T>) -> slice<&T>

    source

    Compiler invokes this function to coerce a mutable slice into a const slice.

    This is necessary, since they are completely different types.
