  • #[cfg] fn main<UserMain>(argc: c_int, argv: &&c_char) -> c_int

    source

    Program entrypoint glue.

    This is the entrypoint for the program. It is called by the C runtime after static initialization. It converts the argc and argv arguments to a slice of strings and calls the user-defined main function.
