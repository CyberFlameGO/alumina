module std::test

Unit testing mini-framework.

This unit testing runner is in `std` for convenience's sake, eventually it should be extracted. Nothing prevents you from writing your own framework, just compile with `--cfg custom_test_framework` and use the test support glue in `std::runtime` to get the test cases.
