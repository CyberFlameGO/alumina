Module test

Unit testing mini-framework.

When --cfg test is provided as an argument to the compiler, this module will override the main entrypoint with the test runner which will run all the test cases defined in the code that is being compiled.

Nothing prevents you from writing your own framework, just compile with --cfg custom_test_framework and use the test support glue in std::runtime to get the test cases.
